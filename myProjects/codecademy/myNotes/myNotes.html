Selectors

This table is organized in specificity. The further down the table you are, the more specific the selector is. More specific selectors override less specific selectors.

Name Example Example's HTML Selected Description
Universal Selector * {} All HTML Selected The universal selector is used to select the entire HTML page.
Tag Selector main {} All Tags Labled "Main" The tag selector is used to select tags on the html page.
Attribute Selector [href] {} All elements containing the href attribute. The Attribute Selector is used to select attributes within an HTML page.

Specific Tag Selector img[src="image.jpg"] {} All image elements with the src attribute linking to image.jpg Used to select a specific element containing a specific tag.
ID Selector #desc {} Selects the element with the "desc" ID attribute. The ID selector is used to select a specific element labled with the corosponding ID attribute.

Flex Boxes

The Flexible box layout, or Flexbox, is a display tool that can be used as a simple way of creating an adaptive website layout.

The Display Property

Setting the display property to flex or inline-flex will set that element to be a flexbox container. Allowing all direct children to be flexbox items. However, containers may also be flex-items themselves, if also nested within a flex-container.

Justifying Content

Justify-content is used to adjust the horizontal position of flexbox items, similar to how text is able to be justified. The Space Around and Between arguments position the items by adding space between or around both sides of them.

Grids

Grids have a similar setup to flexboxes. When you set the display property to grid, or inline-grid, that element becomes a grid container, and its children will be grid items.

Property Arguments Example Description
grid-template row variables / column variables .grid {grid-template: 20px 50% 300px / 30% 20px 500px;} This combines 'grid-template-row' and 'grid-template-column' together as a single property. This allows you to set the size and number of columns and rows.
grid-template-areas names grid-template-areas: "header header"; Allows you to name sections of your webpage that can be used to assign them to parts of the grid.
