Design Guide

Colors!

Webpage

Banner Brown, #b6a594

To be used for the top banner.

Dark Brown, #242320

To be used for the website's background. And for code text background.

Light Brown, #4a4841

To be used for background of text boxes, articles, and wherever found necessary.

Text

Black, #000

To be used for plain text, when obstructed. As well as borders.

White, #ffffff

To be used for plain text. As well as borders.

Light Grey, #d3d3d3

To be used for additional text.

Light Green, #93ed74

To be used for links.

Light Yellow, #edda74

To be used for active links.

Light Purple, #aa74ed

To be used for visited links.

Fonts!

Itim

For the website's Banner Title.

As well as to add flair to any text when wanted.

Pt Mono

To be used with sections of text containing code. Like:
.code {
margin-left: 4px;
color: white;
background-color:dimgrey;
font-family: "Pt Mono";
}

Roboto

This font is for the vast majority of the page. Comes in: Regular. Italic. Bold.

As well as this light form, which includes italics as well!

Text Styles!

H1: The Page's Main Heading

For the title of the webpage only. Optional.

  • font-weight: 700;

  • font-family: 'Roboto';
    • or
  • font-family: 'Itim';

H2: Headings and Titles.

For things like titles of paragraphs.

  • font-family: Roboto;

H3-H6: Sub Headings and Titles.

Used if a larger section is made up of smaller parts that can use distinct titles IE, tables, images, articles.

  • font-family: Roboto;

P: Paragraphs.

For the majority of text found on the website.

  • font-family: Roboto;

.Code: Text including code.

Along with other CSS styling, this is used to mark code found on the page to make it distinct.

  • font-family: "PT Mono";
a: Links.

For links found on the webpage, not including navLinks in the header.

  • color: #93ed74;
